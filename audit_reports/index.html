Quick guide

  1. Pick your name, then choose duplicate mode or Review All Clinics.
  2. Use Keep or Remove on each card, and use the ? button when a clinic needs research.
  3. Tap New list any time to refresh based on urgency mode.
  4. Use Reset to clear only the current review item.
